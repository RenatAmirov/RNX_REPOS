# HaskellDemo

Демонстрационное консольное приложение на Haskell, показывающее работу функций, операторов, полиморфных типов и классов типов.

## Требования

- **Windows 11** (также работает на Linux/macOS)
- Установленный [Stack](https://docs.haskellstack.org/en/stable/install_and_upgrade/) (рекомендуется) или GHC (если хотите компилировать вручную).

## Сборка и запуск

### Способ 1: с использованием Stack (рекомендуется)
